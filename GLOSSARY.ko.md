# Korean Glossary

## Core Terms

- Embedded Rust: 임베디드 러스트
- Bare Metal: 베어메탈
- Microcontroller: 마이크로컨트롤러
- Memory-mapped I/O: 메모리 매핑 I/O
- Peripheral: 주변장치
- Interrupt: 인터럽트
- Exception: 예외
- Cross Compilation: 크로스 컴파일
- Toolchain: 도구 체인
- Flashing: 플래시(쓰기)
- Debugging: 디버깅
- HAL (Hardware Abstraction Layer): HAL(하드웨어 추상화 계층)
- Singleton: 싱글턴
- Typestate: 타입 상태(typestate)
- Zero-cost abstraction: 제로 코스트 추상화

## Style Notes

- Keep crate names and CLI commands in English.
- Keep code identifiers unchanged.
- Prefer one Korean term per concept unless a chapter needs explicit alternatives.
