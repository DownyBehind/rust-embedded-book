# 시작하기

이 섹션에서는 임베디드 프로그램을 작성하고, 빌드하고, 플래시하고,
디버깅하는 과정을 단계별로 안내합니다. 널리 사용되는 오픈소스 하드웨어
에뮬레이터인 QEMU를 활용해 기본 개념을 설명하므로, 대부분의 예제는
별도 하드웨어 없이도 시도할 수 있습니다. 실제 하드웨어가 필요한 유일한
부분은 [하드웨어](./hardware.md) 섹션이며, 여기서는 OpenOCD를 사용해
[STM32F3DISCOVERY] 보드에 프로그램을 올립니다.

[STM32F3DISCOVERY]: http://www.st.com/en/evaluation-tools/stm32f3discovery.html
